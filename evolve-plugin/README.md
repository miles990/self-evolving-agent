# evolve Plugin

> Self-Evolving Agent - 讓 AI 自主達成目標、從經驗中學習並持續改進

## 安裝

```bash
/plugin install miles990/evolve-plugin
```

## 使用

```bash
# 自我進化模式
/evolve [目標描述]

# 建立新 Skill
/evolve --new-skill [skill 名稱]
```

## 功能

- **自我進化**：PDCA 循環 + CheckPoint 護欄 + Memory 系統
- **Skill Creator**：引導式建立 + 範本 + 驗證 + 發布到 GitHub
