# Checkpoint 3: Milestone 完成後 - 目標確認

> 🚨 **強制檢查點** - 不可跳過

## 規則

```
┌─────────────────────────────────────────────────────────────────┐
│  🎯 Milestone 完成後（強制）                                    │
│                                                                 │
│  每個 Milestone 完成時，必須回答三個問題：                      │
│                                                                 │
│  1. 現在的目標是什麼？                                          │
│     → 重新確認當前要達成的目標                                  │
│                                                                 │
│  2. 方向有沒有偏離目標？                                        │
│     → 檢視已完成的工作是否朝向目標前進                          │
│                                                                 │
│  3. 下一步是什麼？                                              │
│     → 明確下一個行動，而非盲目繼續                              │
│                                                                 │
│  ❌ 禁止：用戶說「繼續」就盲目繼續                              │
│  ❌ 禁止：完成後直接跳到下一個而不回顧                          │
│  ✅ 必須：主動報告進度並等待確認                                │
└─────────────────────────────────────────────────────────────────┘
```

## 為什麼重要

防止「目標漂移」- 執行過程中不知不覺偏離原始目標
