## 四大能力

| 能力 | 說明 |
|------|------|
| **Self-Evolution** | 累積可重用的 patterns |
| **Self-Correction** | 失敗時自動修正策略 |
| **Self-Validation** | 驗證學到的知識是否有效 |
| **Personalization** | 適應專案特定風格 |

## 進化流程

```

    ↑                                              │
    └──────────── 失敗修正 ←───────────────────────┘
```
