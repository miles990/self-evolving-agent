# Galgame R15_MAX 福利場景描寫指南

> 基於 galgame-master skill 的福利場景設計系統

## 什麼是「福利場景」

福利場景（ファンサービス / Fan Service）是 Galgame 中用於提升角色魅力、強化玩家體驗的特殊場景。不同於親密場景著重情感連結，福利場景著重於：

- **視覺吸引力**：角色的魅力展現
- **情境設計**：自然而不刻意的安排
- **角色反差**：與日常不同的一面
- **氛圍營造**：曖昧、緊張、心跳加速

---

## R15_MAX 福利場景原則

### 允許 vs 禁止

| 允許 | 禁止 |
|------|------|
| 泳裝、浴衣、睡衣等服裝 | 超過泳裝程度的裸露 |
| 曖昧的身體接近 | 明確的身體接觸描寫 |
| 角色的害羞反應 | 露骨的挑逗對話 |
| 暗示性的視角描寫 | 直接描述身體部位 |
| 意外的尷尬場面 | 刻意的色情場面 |

### 核心原則

```
暗示 > 直接
氛圍 > 畫面
反應 > 事件
留白 > 填滿
```

## 福利場景類型分類

### 1. 服裝類福利

| 場景類型 | 情境 | 重點 |
|----------|------|------|
| **泳裝回** | 海邊、泳池 | 選泳裝的害羞、防曬互動 |
| **浴衣回** | 祭典、溫泉旅行 | 穿浴衣的笨拙、煙火約會 |
| **睡衣回** | 過夜、合宿 | 睡眼惺忪的無防備 |
| **體操服** | 體育課、社團活動 | 運動後的疲憊美 |
| **女僕裝** | 文化祭、打工 | 服務時的害羞 |
| **護士服** | 保健室、cosplay | 照顧時的近距離 |

### 2. 情境類福利

| 場景類型 | 情境 | 重點 |
|----------|------|------|
| **意外撞見** | 更衣室、浴室 | 尷尬的對視、慌張反應 |
| **被困密室** | 倉庫、電梯 | 不得不靠近、心跳加速 |
| **跌倒壓上** | 走廊、教室 | 意外的近距離、僵住 |
| **雨天避雨** | 公車站、屋簷下 | 衣服濕透、共用外套 |
| **睡著靠肩** | 電車、教室 | 無意識的依靠、呼吸聲 |

### 3. 身體接近類福利

| 場景類型 | 情境 | 重點 |
|----------|------|------|
| **擦防曬** | 海邊 | 背部、肩膀的觸碰 |
| **綁頭髮** | 日常 | 後頸的脆弱感 |
| **量身高** | 保健室 | 站得很近、呼吸交錯 |
| **包紮傷口** | 受傷後 | 溫柔的手、低頭專注 |
| **耳語** | 需要小聲說話時 | 呼吸拂過耳朵 |

### 4. 特殊狀態類福利

| 場景類型 | 情境 | 重點 |
|----------|------|------|
| **微醺** | 派對（誤食酒精） | 失去平時的防備 |
| **發燒** | 生病照顧 | 脆弱、依賴、臉紅 |
| **噩夢** | 深夜驚醒 | 尋求安慰、緊抓不放 |
| **催眠** | 劇情需要 | 說出平時不會說的話 |

## 描寫系統設計

### 視角鏡頭語言

```yaml
shot_types:
  # 遠景 - 建立情境
  establish:
    purpose: 展示場景、環境氛圍
    example: "夕陽將海水染成金橙色，沙灘上零星幾個人影。"

  # 中景 - 展示動態
  medium:
    purpose: 角色的整體姿態、動作
    example: "美月小心翼翼地走進水裡，泳裝的緞帶在風中飄動。"

  # 近景 - 強調細節
  close:
    purpose: 表情、局部細節
    example: "她低著頭，耳朵紅得像熟透的蘋果。"

  # 特寫 - 關鍵元素
  detail:
    purpose: 強調重點物件或反應
    example: "一滴水珠從她的鎖骨滑落，消失在泳裝的邊緣。"

  # POV - 主角視角
  pov:
    purpose: 玩家代入感
    example: "她突然抬頭，視線正好和你對上。"
```

### 氛圍層次

```yaml
atmosphere_layers:
  # 環境層
  environment:
    - 光線（夕陽、月光、水面反射）
    - 聲音（海浪、蟬鳴、心跳）
    - 溫度（炎熱、涼爽、溫暖）
    - 氣味（海風、花香、沐浴乳）

  # 角色層
  character:
    - 服裝細節（飄動、濕透、凌亂）
    - 身體反應（臉紅、顫抖、迴避）
    - 表情變化（害羞、緊張、期待）
    - 聲音變化（結巴、壓低、顫抖）

  # 心理層
  psychology:
    - 主角內心OS
    - 角色的心理活動暗示
    - 空氣中的緊張感
    - 時間彷彿靜止
```

### 反應詞彙庫

| 反應類型 | 詞彙選項 |
|----------|----------|
| **臉紅** | 泛紅、染上紅暈、紅透、燒起來、熱度 |
| **迴避** | 別過頭、移開視線、低下頭、轉身、躲閃 |
| **緊張** | 僵住、動彈不得、屏住呼吸、心跳漏拍、腦袋空白 |
| **害羞** | 手足無措、不知所措、結巴、聲音變小、藏起來 |
| **掩飾** | 故作鎮定、假裝沒事、嘴硬、強撐、逞強 |

## R15_MAX 福利場景寫作技巧

### 暗示性描寫技巧

| 技巧 | 範例 |
|------|------|
| **用環境暗示** | 「水珠沿著窗戶滑落」暗示沿著肌膚滑落 |
| **用聲音暗示** | 「聽到她的呼吸聲變重」 |
| **用反應暗示** | 「她的肩膀顫了一下」 |
| **用對話暗示** | 「不要......一直看」 |
| **用時間暗示** | 「時間彷彿靜止了」 |

### 視線引導技巧

```
正確：「視線不自覺地被吸引」
錯誤：「他盯著她的 OOO」

正確：「她用手遮住，但遮不住紅透的臉」
錯誤：「她露出了 OOO」

正確：「衣服勾勒出柔和的線條」
錯誤：「看到了 OOO 的形狀」
```

### 禁止詞彙

以下詞彙在 R15_MAX 中**禁止使用**：

- 任何身體部位的直接稱呼（除了臉、手、肩膀、背、腿等可見部位）
- 露骨的動作描述
- 挑逗性的台詞
- 任何可能聯想到性行為的詞彙

### 替代表達

| 禁止 | 替代 |
|------|------|
| 裸露 | 曲線、輪廓、線條 |
| 看到了 OOO | 視線不知道該放哪裡 |
| 身體接觸 | 溫度、觸感、心跳 |
| 想要 OOO | 想更靠近、想繼續 |
