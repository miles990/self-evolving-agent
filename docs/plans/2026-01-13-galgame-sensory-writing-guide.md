# Galgame R15_MAX 感官描寫指南

> 聲音與氣味的沉浸式寫作技巧

## 核心原則

| 原則 | 說明 |
|------|------|
| **暗示 > 直接** | 用環境音襯托，而非直接描寫 |
| **氛圍 > 細節** | 營造整體感受，而非逐一列舉 |
| **情感 > 肉體** | 聲音與氣味要連結到情感記憶 |
| **留白 > 填滿** | 讓玩家自行想像補完 |

---

# 第一部分：聲音描寫系統

## 1. 聲音類型分類

### 1.1 環境音（氛圍營造）

| 類型 | 範例 | 情緒效果 |
|------|------|----------|
| **雨聲** | 淅淅瀝瀝、嘩啦嘩啦、滴答滴答 | 親密、憂鬱、安心 |
| **風聲** | 呼嘯、輕拂、沙沙 | 孤寂、溫柔、緊張 |
| **蟲鳴** | 唧唧、嗡嗡、蛐蛐 | 夏夜、寧靜、時間流逝 |
| **時鐘** | 滴答、噹噹、嘀嗒 | 緊張、等待、時間停滯 |
| **煙火** | 砰砰、轟隆、嘶嘶 | 浪漫、回憶、瞬間 |

### 1.2 人物聲音（情感表達）

| 類型 | 描寫詞彙 | 適用情境 |
|------|----------|----------|
| **呼吸** | 輕淺、急促、顫抖、平穩、沉重 | 緊張、興奮、安心 |
| **心跳** | 砰砰、怦怦、撲通、狂跳、漏拍 | 心動、緊張、期待 |
| **嘆息** | 輕嘆、長嘆、滿足、無奈、釋然 | 感動、放鬆、認命 |
| **笑聲** | 咯咯、噗哧、輕笑、微笑、苦笑 | 開心、害羞、無奈 |
| **哭泣** | 啜泣、嗚咽、抽噎、淚光、哽咽 | 感動、委屈、幸福 |

### 1.3 動作聲音（暗示性）

| 動作 | 聲音詞彙 | R15_MAX 用法 |
|------|----------|--------------|
| **衣物** | 窸窣、沙沙、滑落 | 暗示而非描述 |
| **接觸** | 輕觸、摩挲、貼近 | 用聲音暗示動作 |
| **床鋪** | 吱嘎、輕響、凹陷 | 環境音代替描寫 |
| **翻身** | 窸窸窣窣、輕輕、悄悄 | 事後場景專用 |

## 2. 聲音描寫模板

### 2.1 黑屏期間的聲音層次

```
【第一層：環境音】
- 設定基調的背景聲音
- 例：雨聲、時鐘聲、遠處的音樂

【第二層：人物音】
- 呼吸、心跳、輕聲細語
- 不直接描寫動作

【第三層：留白】
- 沉默、省略號
- 讓想像填補空白
```

### 2.2 範例模板

**緊張時刻**
```
*房間裡只剩下兩人的呼吸聲*
*窗外的雨聲彷彿隔絕了整個世界*
*能聽到彼此的心跳，砰砰作響*
```

**親密時刻（黑屏）**
```
*沉默*
*只有窗外的雨聲*
*和逐漸靠近的呼吸*
```

**事後時刻**
```
*呼吸漸漸平復*
*時鐘的滴答聲重新變得清晰*
*窗外的鳥鳴，天已經亮了*
```

## 3. 角色專屬聲音設計

### 3.1 傲嬌（美月）

| 情境 | 聲音特徵 |
|------|----------|
| 害羞時 | 聲音突然變小、尾音顫抖、偶爾結巴 |
| 嘴硬時 | 音調升高、語速加快、突然停頓 |
| 認真時 | 聲音堅定但輕柔、沒有了平時的刺 |
| 撒嬌時 | 悶悶的、嘟囔、帶著鼻音 |

**範例**
```
**美月**
「才、才不是......」
*聲音越來越小，最後幾乎聽不見*
*只剩下急促的呼吸聲*
```

### 3.2 冷嬌（凜）

| 情境 | 聲音特徵 |
|------|----------|
| 平時 | 聲音低沉平穩、語速慢、停頓多 |
| 動搖時 | 聲音輕微顫抖、語速不穩 |
| 認真時 | 比平時稍微大聲、每個字都很清晰 |
| 幸福時 | 聲音變軟、帶著微笑的語氣 |

**範例**
```
**凜**
「......」
*沉默了很久*
*當她再次開口時，聲音比平時柔軟了許多*
「......幸福。」
```

### 3.3 小惡魔（響）

| 情境 | 聲音特徵 |
|------|----------|
| 撩人時 | 刻意壓低的性感語調、拉長尾音 |
| 被反撩時 | 聲音突然正常、語速變快、結巴 |
| 認真時 | 完全不同的純淨聲音、沒有任何修飾 |
| 撒嬌時 | 小聲、帶著哭腔、孩子氣 |

**範例**
```
**響**
「學長～想我了......」
*突然，她的聲音變了*
*沒有了平時的撩人語調*
*只剩下顫抖的、真實的聲音*
「......我怕。」
```

## 4. 聲音描寫禁忌

| ❌ 禁止 | ✅ 替代方案 |
|--------|------------|
| 直接描寫行為的聲音 | 用環境音暗示 |
| 露骨的擬聲詞 | 用呼吸、心跳代替 |
| 過度詳細的聲音描寫 | 留白讓玩家想像 |
| 不雅的聲音詞彙 | 用情緒詞彙代替 |

# 第二部分：氣味描寫系統

## 1. 氣味類型分類

### 1.1 環境氣味（場景氛圍）

| 類型 | 範例 | 情緒效果 |
|------|------|----------|
| **雨後** | 泥土的清香、潮濕的空氣、清新 | 淨化、新開始、安心 |
| **夜晚** | 夜風的涼意、月光的清冷（通感） | 浪漫、神秘、寧靜 |
| **室內** | 咖啡香、書香、木質調 | 舒適、日常、安全 |
| **季節** | 櫻花香、夏草味、落葉氣息 | 時間感、回憶、變化 |

### 1.2 人物氣味（親密標記）

| 類型 | 描寫詞彙 | 適用角色 |
|------|----------|----------|
| **髮香** | 淡淡的、清香、熟悉的 | 所有女性角色 |
| **體溫** | 溫暖的、微熱的（通感） | 親密場景 |
| **香水** | 淡雅的、若有似無的 | 大小姐、偶像 |
| **洗衣精** | 陽光味、清新、乾淨 | 日常系角色 |
| **專屬味道** | 獨特的、只屬於她的 | 建立情感連結 |

### 1.3 情緒氣味（氛圍強化）

| 情緒 | 關聯氣味 | 用法 |
|------|----------|------|
| **緊張** | 空氣變得稀薄、呼吸中的微熱 | 黑屏前 |
| **安心** | 熟悉的味道、溫暖的氣息 | 事後 |
| **心動** | 靠近時的香氣、呼吸的溫度 | 告白 |
| **幸福** | 陽光的味道、混合的氣息 | 結局 |

---

## 2. 氣味描寫模板

### 2.1 親密場景的氣味層次

```
【第一層：環境氣味】
- 房間、季節、時間的味道
- 例：雨後的清新、夜晚的涼意

【第二層：人物氣味】
- 髮香、體溫、專屬味道
- 建立情感連結

【第三層：混合氣味】
- 兩人的氣息交融
- 用「我們」代替個體
```

### 2.2 範例模板

**靠近時**
```
*她靠近時，熟悉的髮香飄入鼻尖*
*淡淡的，像是春天的花朵*
*這是只屬於她的味道*
```

**黑屏期間（畫外音）**
```
**她**（畫外音）
「......你身上的味道......」
*聲音很輕*
「......讓我很安心。」
```

**事後**
```
*清晨的陽光帶著溫暖的氣息*
*混雜著兩人的體溫、汗水，還有......*
*一種難以言喻的，幸福的味道*
```

## 3. 角色專屬氣味設計

### 3.1 傲嬌青梅竹馬 — 美月

```yaml
character_scent:
  name: 美月
  base_note: 柑橘調（清新、活潑）
  heart_note: 白花香（女孩子氣、純潔）
  special: 洗衣精的陽光味（青梅竹馬的熟悉感）

  scene_usage:
    日常: "她身上總是有淡淡的柑橘香，像是剛洗完的衣服"
    靠近: "靠近時，熟悉的香氣變得更加濃郁......那是從小就聞慣的味道"
    親密: "她的髮絲貼在他臉上，那股熟悉的香氣讓他想起無數個共同的夏天"
```

### 3.2 冷嬌學姐 — 凜

```yaml
character_scent:
  name: 凜
  base_note: 墨香、紙張（書卷氣）
  heart_note: 淡淡的白茶（清冷、知性）
  special: 舊書的味道（圖書室的記憶）

  scene_usage:
    日常: "她身上總有一股淡淡的書香，像是把圖書室的氣息帶在身上"
    靠近: "靠近時，能聞到紙張和淡茶的清香......還有她獨有的、冷冽的氣息"
    親密: "她的長髮散落，帶著墨香和體溫混合的味道......那是只屬於他的秘密"
```

### 3.3 小惡魔學妹 — 響

```yaml
character_scent:
  name: 響
  base_note: 甜美的果香（偶像的華麗）
  heart_note: 若有似無的香水（刻意營造）
  special: 卸下偽裝後，是洗髮精的清新味道（真實的她）

  scene_usage:
    日常: "她走過時會留下一陣若有似無的甜香......刻意設計的，讓人移不開注意力"
    靠近: "靠近時，那股甜膩的香氣變淡了......取而代之的是洗髮精的清新"
    親密: "沒有了香水的偽裝，只剩下最真實的味道......乾淨的，純粹的，像是另一個人"
```

## 4. 感官描寫整合範例

### 範例一：雨夜的親密（美月路線）

**【黑屏前】**

```
*雨聲嘩啦啦地打在窗上*
*房間裡瀰漫著雨水的清新氣息*

**美月**
「......」
*她的呼吸聲在安靜的房間裡格外清晰*
*能聞到她髮絲間熟悉的柑橘香*
*還有......因為淋雨而更加濃郁的、專屬於她的味道*

**美月**
「......陽太。」
*聲音很小，帶著顫抖*
「......靠近一點。」

*兩人之間的距離縮短*
*呼吸聲交疊在一起*
*她的體溫帶著雨水的涼意......卻又溫暖得讓人心跳加速*
```

**【黑屏期間】**

```
【畫面轉黑】

*雨聲*
*心跳聲*
*交織在一起的呼吸*

---

**美月**（畫外音）
*聲音悶悶的，帶著顫抖*
「......你身上的味道......」
「......讓我想起小時候。」

**陽太**（畫外音）
「什麼味道？」

**美月**（畫外音）
「......說不上來。」
*聲音越來越小*
「......但是很安心。」

*沉默*
*只剩下雨聲和彼此的呼吸*

---

*窸窣聲*
*心跳聲加速*

**美月**（畫外音）
*帶著哭腔*
「......抱緊我。」

*雨聲變大*
*彷彿要遮蓋一切*
```

**【事後】**

```
【畫面漸亮】

*雨已經小了*
*房間裡瀰漫著兩人混合的氣息*
*雨水的清新、她的柑橘香、還有溫暖的體溫......*
*交織成一種難以言喻的味道*

**美月**
*縮在他懷裡，聲音帶著滿足的慵懶*
「......還在下雨。」

**陽太**
「嗯。」

**美月**
*深深吸了一口氣*
「......你的味道。」
*聲音悶悶的*
「......我想記住。」

**陽太**
「以後每天都能聞到。」

**美月**
*輕輕打了他一下*
「......笨蛋。」
*但聲音帶著笑意*
「......說話算話喔。」

*窗外，雨聲漸漸變小*
*空氣中瀰漫著雨後的清新*
*還有......幸福的味道*
```

## 5. 感官描寫詞彙速查表

### 聲音詞彙

| 類別 | 詞彙 |
|------|------|
| **呼吸** | 輕淺、急促、顫抖、平穩、灼熱、交織、靠近、貼近 |
| **心跳** | 砰砰、怦怦、撲通、漏拍、加速、同步、共鳴 |
| **聲線** | 顫抖、沙啞、哽咽、輕柔、堅定、破碎、融化 |
| **環境** | 滴答、沙沙、窸窣、嘩啦、寂靜、迴響 |

### 氣味詞彙

| 類別 | 詞彙 |
|------|------|
| **基調** | 清新、溫暖、甜美、淡雅、冷冽、熟悉 |
| **髮香** | 洗髮精、花香、陽光味、專屬的、獨特的 |
| **體溫** | 溫熱、微燙、安心、熟悉、交融、混合 |
| **環境** | 雨後、夜風、晨光、咖啡香、書香、花香 |

### 通感詞彙（跨感官）

| 組合 | 範例 |
|------|------|
| 聲音→觸感 | 「她的聲音像絲綢一樣滑過耳際」 |
| 氣味→溫度 | 「她的香氣帶著溫暖的氣息」 |
| 心跳→音樂 | 「兩人的心跳像是在合奏」 |
| 呼吸→風 | 「她的呼吸輕輕拂過，像春風」 |

## 6. 禁忌與替代

| ❌ 禁止 | ✅ 替代方案 |
|--------|------------|
| 直接描寫不雅聲音 | 用環境音（雨聲）掩蓋 |
| 描寫汗味等不雅氣味 | 用「混合的氣息」「溫暖的味道」 |
| 過於詳細的感官細節 | 保持朦朧，讓想像補完 |
| 色情暗示的擬聲詞 | 用呼吸聲、心跳聲代替 |
