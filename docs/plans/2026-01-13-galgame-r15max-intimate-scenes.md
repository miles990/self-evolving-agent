# Galgame R15_MAX 親密互動場景集

> 基於 galgame-master skill 的 R15_MAX 分級親密場景範例

## R15_MAX 呈現原則

| 項目 | 允許 | 禁止 |
|------|------|------|
| **暗示方式** | 「過夜」「待到早上」「更近一點」 | 明說行為細節 |
| **呈現手法** | 黑屏 + 畫外音對話 + 事後 CG | 過程中的視覺 |
| **畫外音** | 情緒對話（緊張、溫柔、依賴） | 露骨台詞 |
| **事後 CG** | 依偎、滿足微笑、亂髮、氛圍 | 裸露超過泳裝 |
| **核心重點** | **情感連結** | 肉體描寫 |

# R15_MAX 寫作技巧總結

## 黑屏期間的對話設計

| 情緒 | 適合台詞 |
|------|----------|
| **緊張** | 「等一下......」「有點害怕......」 |
| **依賴** | 「抱緊一點......」「不要停......」 |
| **確認** | 「你是我的......」「只看著我......」 |
| **感動** | 「很幸福......」「想哭......」 |
| **撒嬌** | 「溫柔一點......」「再一次......」 |

## 事後場景要點

| 要素 | 呈現方式 |
|------|----------|
| **視覺** | 散亂的髮絲、依偎的姿態、柔和的光線 |
| **對話** | 低語、撒嬌、告白、約定 |
| **情感** | 滿足、安心、堅定、感動 |
| **留白** | 不描述過程，只呈現結果的氛圍 |

## 不同角色類型的 R15_MAX 特色

| 類型 | 特色 |
|------|------|
| **傲嬌** | 嘴硬但身體誠實、事後害羞道歉、用「笨蛋」表達愛 |
| **冷嬌** | 話少但行動多、罕見的主動、用「幸福」表達感謝 |
| **小惡魔** | 獵人變獵物、卸下偽裝、用「認真」表達心意 |
| **病弱** | 珍惜當下、脆弱中的堅強、用「活著」表達感激 |
